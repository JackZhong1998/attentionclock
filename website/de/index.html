Installation
  1. Heruntergeladene .dmg doppelklicken
  2. Doppelklick auf Installieren und öffnen
  3. Click Done if blocked
  4. System Settings → Open Anyway for Installieren und öffnen
  5. Double-click Installieren und öffnen again
  6. Click Cancel
  7. System Settings → Open Anyway for Attention Clock
  8. Open the app again
