Installation
  1. Double-cliquez sur le .dmg téléchargé
  2. Double-cliquez sur Installer et ouvrir
  3. Double-click Installer et ouvrir in the DMG
  4. If macOS says it cannot verify “Installer et ouvrir”, click Done
  5. Open System Settings → Privacy & Security (or double-click Open System Settings.webloc)
  6. Under Security, click Open Anyway for Installer et ouvrir
  7. Double-click Installer et ouvrir again — the install guide dialog will appear and install Attention Clock
  8. Order matters: allow Install and Open first, or the guide dialog will not appear.
  9. If you see damaged, click Cancel (not Move to Trash)
  10. Open System Settings → Privacy & Security
  11. Under Security, click Open Anyway for Attention Clock
  12. Open Attention Clock again
