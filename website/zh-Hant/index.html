適用於 macOS 14.0 及以上，完全免費。請選擇晶片類型。

下載 .dmg GitHub 所有版本
安裝步驟
  1. 雙擊下載的 .dmg 檔案
  2. 雙擊 安裝並開啟
  3. 無法驗證時點 完成
  4. 系統設定 → 為 安裝並開啟 點 仍要打開
  5. 再次按兩下 安裝並開啟
  6. 點 取消
  7. 系統設定 → 仍要打開
  8. 再次開啟
