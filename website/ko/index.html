macOS 14.0 이상, 무료. 칩을 선택하세요.

.dmg 다운로드 GitHub 모든 릴리스
설치 방법
  1. 다운로드한 .dmg 더블클릭
  2. 응용 프로그램으로 드래그
  3. 디스크 이미지 꺼내기
  4. 첫 실행 차단 시: 시스템 설정 → 개인정보 보호 및 보안 → 그래도 열기
