**專注時鐘**是一款免費、開源的 Mac 桌面應用。幫你設定一段專注時間（預設 25 分鐘），安靜做事；時間到了會提醒你，並記錄每天的成果。

不需要註冊帳號，不需要連網，所有資料都保存在你自己的 Mac 上。

## 好在哪裡？

- **打開就能用**：點「開始專注」即可
- **介面乾淨**：大圓環倒數、留白充足
- **時間靈活**：預設 25 分鐘，可 ±5 分鐘調整
- **隨時掌控**：支援暫停、繼續、提前結束
- **看見進步**：每完成一次種一棵樹 🌳，還有熱力圖
- **桌面夥伴（可選）**：3000+ 像素角色，可下載、可浮窗常駐桌面
- **完全免費**：開源專案，程式碼公開

## 桌面夥伴（可選）

專注不必一個人硬撐。開啟桌面夥伴後，可以：

- **3000+ 像素圖鑑**：瀏覽、搜尋、下載喜歡的角色或 IP
- **專注時有回應**：夥伴陪在計時器旁，表情隨狀態變化
- **桌面浮窗常駐**：把夥伴放在桌面上，切換應用也不離開
- **完全可選**：隨時關閉，不影響計時與統計

## 怎麼用？

1. 下載並安裝應用
2. 在「應用程式」中開啟 **專注時鐘**
3. 點 **開始專注** 開始倒數
4. 結束後會收到完成提醒

## 安裝

1. 雙擊 `.dmg` 檔案
2. 雙擊「安裝並開啟」自動安裝並啟動
3. 若提示「已損壞」，請看《首次開啟指南》

### 按「安裝並開啟」顯示無法驗證？

從瀏覽器下載後，**安裝助手本身**也可能先被 macOS 攔截。系統彈窗通常只有 **「完成」**，**不會**出現我們的安裝引導。

1. 按兩下 DMG 中的 **「安裝並開啟」**
2. 若顯示 **Apple 無法驗證…**，點 **完成**
3. 開啟 **系統設定** → **隱私權與安全性**
4. 在 **安全性** 為 **「安裝並開啟」** 點 **仍要打開**
5. **再次按兩下「安裝並開啟」** — 此時才會出現安裝引導彈窗

> **順序很重要**：必須先為安裝助手點「仍要打開」。

### 安裝後開啟專注時鐘顯示「已損壞」？

安裝助手成功後會嘗試開啟專注時鐘。若被攔截，請依序操作：

1. 若彈出 **「已損壞」**，點 **取消**
2. 開啟 **系統設定** → **隱私權與安全性**
3. 在 **安全性** 為 **專注時鐘** 點 **仍要打開**
4. 再次開啟應用

> **順序很重要**：必須先觸發一次攔截。

## 常見問題

**安裝或開啟時被攔截？**  
先為安裝助手點仍要打開，再為專注時鐘點仍要打開。

**需要連網嗎？**  
不需要，完全在本機運行。

**資料存在哪？**  
存在你的 Mac 本機，不會上傳。

**支援 Windows 嗎？**  
目前僅支援 Mac（macOS 14.0+）。
