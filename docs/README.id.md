**Jam Fokus** is a free, open-source Mac focus timer. Set a session (default 25 min), work without distractions, and track your daily progress.

No account or internet required. All data stays on your Mac.

## Why you'll like it

- **One tap to start**
- **Clean, calm design**
- **Flexible timing** (default 25 min)
- **Pause, resume, or end anytime**
- **Track progress** with trees 🌳 and a heatmap
- **Optional pixel cat companion**
- **Completely free** and open source

## How to use

1. Download and install
2. Open **Jam Fokus** from Applications
3. Tap **Start Focus**
4. Get a notification when done

## Install

1. Open the `.dmg` file
2. Drag **Jam Fokus** to Applications
3. Eject and launch the app

### Peringatan «rusak» saat pertama dibuka?

macOS dapat memblokir app unduhan di luar App Store. **App tidak benar-benar rusak.** Ikuti langkah **berurutan**:

1. Di **Aplikasi**, **klik dua kali** Attention Clock
2. Saat dialog «rusak» muncul, klik **Batal** (⚠️ **jangan** **Pindahkan ke Tong Sampah**)
3. Buka **Pengaturan Sistem** → **Privasi & Keamanan**
4. Gulir ke **Keamanan** dan klik **Buka Tetap** untuk Attention Clock
5. Buka app lagi

> **Urutan penting**: langkah 1 dan 2 harus dulu, baru muncul **Buka Tetap**.

## FAQ

**Muncul «rusak»?**  
Ikuti langkah: klik dua kali → Batal → Buka Tetap di Pengaturan.

**Internet required?**  
No, runs entirely on your Mac.

**Where is data stored?**  
Locally on your Mac only.

**Windows or iPhone?**  
Mac only, macOS 14.0+.
