**Fokus-Uhr** ist eine kostenlose Open-Source-Mac-App. Sie hilft dir, dich zu konzentrieren (Standard 25 Minuten) und zeichnet deinen täglichen Fortschritt auf.

Kein Konto, kein Internet nötig. Alle Daten bleiben auf deinem Mac.

## Das spricht dafür

- **Sofort startklar**: öffnen und Fokus starten
- **Klares Design**: großer Fortschrittsring
- **Flexible Zeit**: 25 Min. Standard, ±5 Min.
- **Volle Kontrolle**: Pause, Fortsetzen, vorzeitig beenden
- **Fortschritt sehen**: ein Baum 🌳 pro Session, Heatmap
- **Desktop-Begleiter (optional)**: 3000+ Pixel-Charaktere, downloadbar mit Schwebefenster
- **Komplett kostenlos**: Open Source

## Desktop-Begleiter (optional)

Fokussieren muss nicht einsam sein. Mit Desktop-Begleitern kannst du:

- **3000+ Katalog**: durchsuchen, suchen und herunterladen
- **Reagiert beim Fokus**: begleitet den Timer
- **Schwebefenster**: auf dem Desktop sichtbar
- **Vollständig optional**: jederzeit abschaltbar

## So funktioniert's

1. App herunterladen und installieren
2. **Fokus-Uhr** aus Programme öffnen
3. **Fokus starten** tippen
4. Benachrichtigung bei Ende

## Installation

1. `.dmg` öffnen
2. **Installieren und öffnen** doppelklicken
3. Bei „beschädigt“ die Anleitung im DMG lesen

### Cannot verify Install and Open?

1. Click **Done** if blocked
2. System Settings → **Open Anyway** for **Installieren und öffnen**
3. Double-click **Installieren und öffnen** again

### Attention Clock “damaged”?

1. Click **Cancel**
2. System Settings → **Open Anyway** for Attention Clock
3. Open the app again

## FAQ
