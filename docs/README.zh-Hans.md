# 专注时钟

[繁體中文](README.zh-Hant.md) · [English](README.en.md) · [日本語](README.ja.md) · [한국어](README.ko.md) · [更多语言](../README.md)

**专注时钟**是一款免费、开源的 Mac 桌面应用。它帮你设定一段专注时间（默认 25 分钟），在这段时间里安心做事；时间到了会提醒你，还会记录你每天的专注成果。还可选 **3000+ 桌面伙伴** 陪你一起专注。

不需要注册账号，不需要联网，所有数据都保存在你自己的 Mac 上。

---

## 好在哪里？

- **打开就能用**：没有复杂设置，点开「开始专注」即可
- **界面干净**：大圆环倒计时、留白充足，不打扰你的注意力
- **时间灵活**：默认 25 分钟，可以按 ±5 分钟快捷调整，也能在设置里改默认时长
- **随时掌控**：支持暂停、继续、提前结束
- **看见进步**：每完成一次专注种一棵树 🌳，还有热力图帮你回顾过去半年的坚持
- **桌面伙伴（可选）**：3000+ 像素角色，可下载、可浮窗常驻桌面
- **完全免费**：开源项目，代码公开，放心使用

---

## 桌面伙伴（可选）

专注不必一个人硬撑。开启桌面伙伴后，可以：

- **3000+ 像素图鉴**：在「桌面伙伴」页浏览、搜索、下载喜欢的角色或 IP
- **专注时有回应**：伙伴陪在计时器旁，表情随专注状态变化，完成时还会庆祝
- **桌面浮窗常驻**：开启「桌面浮窗」，伙伴在桌面上可见，切换应用也不离开
- **完全可选**：随时关闭，不影响计时与统计

## 怎么用？

### 第一次使用

1. 按下方说明下载并安装应用
2. 在「应用程序」文件夹里打开 **专注时钟**
3. 点击 **开始专注**，圆环开始倒计时
4. 专注结束后，系统会弹出完成提醒

### 日常使用

| 你想做的事 | 怎么操作 |
|------------|----------|
| 开始专注 | 点击「开始专注」 |
| 调整本次时长 | 在倒计时圆环两侧点 **−** 或 **+**（每次 5 分钟） |
| 中途休息 | 点击「暂停」，休息完点「继续」 |
| 提前结束 | 点击「结束」 |
| 挑选桌面伙伴 | 打开「桌面伙伴」→ 浏览图鉴 → 下载并选择 |
| 开启桌面浮窗 | 「桌面伙伴」→ 打开「桌面浮窗」 |
| 修改默认时长 | 打开「设置」→ 调整「默认时长」 |
| 查看统计 | 打开「统计」→ 看今日数据、热力图、历史记录 |

### 四个主要页面

- **专注**：倒计时和操作按钮，已选伙伴会显示在下方
- **桌面伙伴**：图鉴浏览、搜索下载、管理已下载角色
- **统计**：今天完成了几次、专注了多久、热力图和每日记录
- **设置**：默认时长、桌面伙伴开关等

## 怎么下载？

### 第一步：确认你的 Mac 芯片类型

1. 点击屏幕左上角 **苹果菜单**
2. 选择 **关于本机**
3. 查看信息：
   - 显示 **芯片**（如 Apple M2）→ 你是 **Apple 芯片**
   - 显示 **处理器**（如 Intel Core i5）→ 你是 **Intel 芯片**

### 第二步：点击对应链接下载

当前最新版本：**1.1.0**

| 芯片类型 | 直接下载（简体中文） |
|----------|----------------------|
| Apple 芯片（M 系列） | [**下载 Apple 芯片版**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.1.0/AttentionClock-1.1.0-zh-Hans-arm64.dmg) |
| Intel 芯片 | [**下载 Intel 版**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.1.0/AttentionClock-1.1.0-zh-Hans-x86_64.dmg) |

也可以前往 **[所有版本下载页](https://github.com/JackZhong1998/attentionclock/releases/latest)**，在列表中找到文件名含 `zh-Hans` 的安装包。

### 第三步：安装

1. 双击下载好的 `.dmg` 文件
2. 双击 **「安装并打开」**

### 第四步：安装助手被拦截（只弹出「完成」、没有安装引导）

从浏览器下载后，**「安装并打开」本身**也可能先被 macOS 拦截。此时系统弹窗里通常只有 **「完成」**，**不会出现**我们的安装引导——这是正常现象。

1. 双击 **「安装并打开」**
2. 若弹出 **「未打开“安装并打开”」/ Apple 无法验证…**，点击 **完成**（不要移到废纸篓）
3. 打开 **系统设置** → **隐私与安全性**（可双击 DMG 内 **「打开系统设置」**）
4. 在 **安全性** 区域找到 **「安装并打开」**，点击 **仍要打开**
5. **再次双击「安装并打开」** — 此时会先出现 **「继续安装」** 引导弹窗，然后自动安装专注时钟

> **顺序很重要**：必须先为「安装并打开」点一次「仍要打开」，安装引导弹窗才会出现。

### 第五步：专注时钟被拦截（提示「已损坏」）

安装助手成功运行后，会复制并尝试打开专注时钟。若此时被拦截：

1. 弹出 **「已损坏」** 时，点击 **取消**
2. 打开 **系统设置** → **隐私与安全性**
3. 在 **安全性** 区域为 **专注时钟 / AttentionClock** 点击 **仍要打开**
4. 再次打开专注时钟

> DMG 内的 **「① 请先阅读.txt」** 含完整图文步骤。

## 常见问题

**需要联网吗？**  
专注计时和统计不需要。浏览、下载桌面伙伴时需要网络。

**数据存在哪里？**  
存在你的 Mac 上（`~/Library/Application Support/AttentionClock/`），不会上传到任何服务器。

**桌面伙伴是什么？**  
可选功能。从 3000+ 像素图鉴中下载角色或伙伴，专注时陪你一起做事，还可开启桌面浮窗。不用也完全不影响计时。

**支持 Windows / iPhone 吗？**  
目前仅支持 Mac（macOS 14.0 及以上）。

**下载链接打不开？**  
请确认已发布 [GitHub Release](https://github.com/JackZhong1998/attentionclock/releases)。若尚未发布，可先查看源码自行编译，或等待作者上传安装包。

**打开时提示「已损坏」或被拦截？**  
分两步：先为「安装并打开」点仍要打开，再为专注时钟点仍要打开。详见上方第四、五步。
