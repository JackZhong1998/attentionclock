# 专注时钟

[繁體中文](README.zh-Hant.md) · [English](README.en.md) · [日本語](README.ja.md) · [한국어](README.ko.md) · [更多语言](../README.md)

**专注时钟**是一款免费、开源的 Mac 桌面应用。它帮你设定一段专注时间（默认 25 分钟），在这段时间里安心做事；时间到了会提醒你，还会记录你每天的专注成果。

不需要注册账号，不需要联网，所有数据都保存在你自己的 Mac 上。

---

## 好在哪里？

- **打开就能用**：没有复杂设置，点开「开始专注」即可
- **界面干净**：大圆环倒计时、留白充足，不打扰你的注意力
- **时间灵活**：默认 25 分钟，可以按 ±5 分钟快捷调整，也能在设置里改默认时长
- **随时掌控**：支持暂停、继续、提前结束
- **看见进步**：每完成一次专注种一棵树 🌳，还有热力图帮你回顾过去半年的坚持
- **可爱陪伴（可选）**：可以开启「云养猫」，一只像素小猫陪你专注；还能把它放到桌面上
- **完全免费**：开源项目，代码公开，放心使用

---

## 怎么用？

### 第一次使用

1. 按下方说明下载并安装应用
2. 在「应用程序」文件夹里打开 **专注时钟**
3. 点击 **开始专注**，圆环开始倒计时
4. 专注结束后，系统会弹出完成提醒

### 日常使用

| 你想做的事 | 怎么操作 |
|------------|----------|
| 开始专注 | 点击「开始专注」 |
| 调整本次时长 | 在倒计时圆环两侧点 **−** 或 **+**（每次 5 分钟） |
| 中途休息 | 点击「暂停」，休息完点「继续」 |
| 提前结束 | 点击「结束」 |
| 修改默认时长 | 打开「设置」→ 调整「默认时长」 |
| 查看统计 | 打开「统计」→ 看今日数据、热力图、历史记录 |
| 开启养猫 | 打开「设置」→ 打开「开始云养猫」 |

### 三个主要页面

- **专注**：倒计时和操作按钮
- **统计**：今天完成了几次、专注了多久、热力图和每日记录
- **设置**：默认时长、云养猫开关等

## 怎么下载？

### 第一步：确认你的 Mac 芯片类型

1. 点击屏幕左上角 **苹果菜单**
2. 选择 **关于本机**
3. 查看信息：
   - 显示 **芯片**（如 Apple M2）→ 你是 **Apple 芯片**
   - 显示 **处理器**（如 Intel Core i5）→ 你是 **Intel 芯片**

### 第二步：点击对应链接下载

当前最新版本：**1.0.0**

| 芯片类型 | 直接下载（简体中文） |
|----------|----------------------|
| Apple 芯片（M 系列） | [**下载 Apple 芯片版**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-zh-Hans-arm64.dmg) |
| Intel 芯片 | [**下载 Intel 版**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-zh-Hans-x86_64.dmg) |

也可以前往 **[所有版本下载页](https://github.com/JackZhong1998/attentionclock/releases/latest)**，在列表中找到文件名含 `zh-Hans` 的安装包。

### 第三步：安装

1. 双击下载好的 `.dmg` 文件
2. 把 **专注时钟** 图标拖到 **应用程序** 文件夹
3. 弹出安装窗口后，推出磁盘映像
4. 在「应用程序」中打开专注时钟

### 首次打开提示「已损坏」或「无法验证开发者」？

这是 macOS 对从浏览器下载、未上架 App Store 应用的常见拦截，**不是应用真的坏了**。

**方法一（推荐，终端一行命令）**

打开「终端」，粘贴运行：

```bash
xattr -cr /Applications/AttentionClock.app
```

然后重新双击打开应用。

**方法二（右键打开）**

1. 在「应用程序」中找到 **AttentionClock**（或 **专注时钟**）
2. **按住 Control 键点击**（或右键）→ 选择 **打开**
3. 在弹窗中再次点击 **打开**

**方法三（系统设置）**

1. 打开 **系统设置** → **隐私与安全性**
2. 若出现「已阻止 AttentionClock…」，点击 **仍要打开**

> **公司电脑注意**：若安全性页面显示「此设置已由描述文件配置」，说明 Mac 由公司统一管理，可能不会出现「仍要打开」按钮。请优先试方法一，或联系 IT 管理员放行该应用。

> **彻底免拦截**：需要开发者使用 Apple 开发者账号对应用进行签名并公证（Notarization）。当前为开源免费分发版本，暂未公证。

## 常见问题

**需要联网吗？**  
不需要。专注时钟完全在本地运行。

**数据存在哪里？**  
存在你的 Mac 上（`~/Library/Application Support/AttentionClock/`），不会上传到任何服务器。

**支持 Windows / iPhone 吗？**  
目前仅支持 Mac（macOS 14.0 及以上）。

**下载链接打不开？**  
请确认已发布 [GitHub Release](https://github.com/JackZhong1998/attentionclock/releases)。若尚未发布，可先查看源码自行编译，或等待作者上传安装包。

**打开时提示「已损坏」？**  
在终端运行：`xattr -cr /Applications/AttentionClock.app`，然后重新打开。详见上方安装说明。
