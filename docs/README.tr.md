**Odak Saati** is a free, open-source Mac focus timer. Set a session (default 25 min), work without distractions, and track your daily progress.

No account or internet required. All data stays on your Mac.

## Why you'll like it

- **One tap to start**
- **Clean, calm design**
- **Flexible timing** (default 25 min)
- **Pause, resume, or end anytime**
- **Track progress** with trees 🌳 and a heatmap
- **Optional pixel cat companion**
- **Completely free** and open source

## How to use

1. Download and install
2. Open **Odak Saati** from Applications
3. Tap **Start Focus**
4. Get a notification when done

## Install

1. Open the `.dmg` file
2. Drag **Odak Saati** to Applications
3. Eject and launch the app

### Blocked on first launch?

**System Settings** → **Privacy & Security** → **Open Anyway**

## FAQ

**Internet required?**  
No, runs entirely on your Mac.

**Where is data stored?**  
Locally on your Mac only.

**Windows or iPhone?**  
Mac only, macOS 14.0+.
