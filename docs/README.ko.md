# 집중 타이머

[简体中文](README.zh-Hans.md) · [繁體中文](README.zh-Hant.md) · [English](README.en.md) · [日本語](README.ja.md) · [다른 언어](../README.md)

**집중 타이머**는 Mac용 무료 오픈소스 집중 앱입니다. 타이머를 설정하고(기본 25분) 한 가지 일에 몰입하세요. 시간이 되면 알려 주고, 매일의 성과를 기록해 줍니다.

계정 가입 불필요. 인터넷 불필요. 모든 데이터는 내 Mac에만 저장됩니다.

---

## 이런 점이 좋아요

- **바로 시작**: 앱을 열고 「집중 시작」만 누르면 됩니다
- **깔끔한 화면**: 큰 카운트다운 링과 넉넉한 여백 — 산만해지지 않아요
- **시간 조절 자유**: 기본 25분, ±5분으로 바꿀 수 있고, 설정에서 기본값도 변경 가능
- **언제든 제어**: 일시정지, 재개, 조기 종료 지원
- **성장이 보여요**: 집중 완료마다 나무 한 그루 🌳. 지난 26주 히트맵으로 꾸준함 확인
- **귀여운 동반자(선택)**: 「클라우드 고양이」를 켜면 픽셀 고양이가 함께합니다. 데스크톱에 띄울 수도 있어요
- **완전 무료**: 오픈소스이며 코드가 공개되어 있습니다

## 사용 방법

### 처음 사용할 때

1. 아래 링크에서 다운로드 후 설치
2. 「응용 프로그램」 폴더에서 **집중 타이머** 실행
3. **집중 시작** 클릭 — 카운트다운 시작
4. 시간이 되면 완료 알림이 표시됩니다

### 일상적으로

| 하고 싶은 것 | 방법 |
|-------------|------|
| 집중 시작 | **집중 시작** 클릭 |
| 이번 시간 조절 | 링 옆 **−** / **+** (5분 단위) |
| 잠깐 쉬기 | **일시정지** → **계속** |
| 미리 끝내기 | **종료** |
| 기본 시간 변경 | **설정** → **기본 시간** |
| 통계 보기 | **통계** → 오늘 기록, 히트맵 |
| 고양이 켜기 | **설정** → **클라우드 고양이 켜기** |

### 세 가지 탭

- **집중**: 타이머와 조작 버튼
- **통계**: 오늘 횟수, 히트맵, 일별 기록
- **설정**: 기본 시간, 클라우드 고양이 등

## 다운로드 방법

### 1단계: Mac 칩 종류 확인

1. 화면 왼쪽 위 **Apple 메뉴** 클릭
2. **이 Mac에 관하여** 선택
3. 확인:
   - **칩**(예: Apple M2) → **Apple 실리콘**
   - **프로세서**(예: Intel Core i5) → **Intel Mac**

### 2단계: 링크를 눌러 다운로드

최신 버전: **1.0.0**

| 칩 | 바로 다운로드 (한국어) |
|----|------------------------|
| Apple 실리콘 (M 시리즈) | [**Apple 실리콘용 다운로드**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-ko-arm64.dmg) |
| Intel | [**Intel용 다운로드**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-ko-x86_64.dmg) |

또는 **[전체 릴리스 페이지](https://github.com/JackZhong1998/attentionclock/releases/latest)**에서 파일명에 `ko`가 포함된 항목을 선택하세요.

### 3단계: 설치

1. 다운로드한 `.dmg` 파일 더블클릭
2. **집중 타이머**를 **응용 프로그램** 폴더로 드래그
3. 디스크 이미지 꺼내기
4. 응용 프로그램에서 실행

### 4단계: 첫 실행（「손상됨」이 뜨는 경우）

브라우저에서 다운로드한 앱은 macOS에서 차단될 수 있습니다. **앱이 실제로 손상된 것은 아닙니다.** 아래 순서를 **반드시 지켜** 주세요:

1. 「응용 프로그램」에서 AttentionClock(집중 타이머)을 **더블 클릭**하여 실행
2. 「손상됨」 대화상자가 뜨면 **취소** 클릭 (⚠️ **휴지통으로 이동** 은 누르지 마세요)
3. **시스템 설정** → **개인 정보 보호 및 보안** 열기
4. **보안** 섹션까지 스크롤하여 AttentionClock의 **확인 없이 열기** 클릭
5. 앱을 다시 열어 정상 사용

> **순서가 중요합니다**: 1·2단계를 먼저 해야 **확인 없이 열기** 버튼이 나타납니다.

## 자주 묻는 질문

**인터넷이 필요한가요?**  
아니요. 완전히 로컬에서 동작합니다.

**데이터는 어디에 저장되나요?**  
내 Mac에만 저장됩니다 (`~/Library/Application Support/AttentionClock/`). 외부로 전송되지 않습니다.

**Windows / iPhone도 되나요?**  
현재 Mac(macOS 14.0 이상)만 지원합니다.

**다운로드 링크가 안 열리면?**  
[GitHub Release](https://github.com/JackZhong1998/attentionclock/releases)가 게시되었는지 확인해 주세요. 아직 없다면 게시를 기다려 주세요.

**열 때 「손상됨」이 뜨나요?**  
위 4단계 순서대로: 더블 클릭 → 취소 → 시스템 설정에서 확인 없이 열기.
