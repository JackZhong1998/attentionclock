# 集中タイマー

[简体中文](README.zh-Hans.md) · [繁體中文](README.zh-Hant.md) · [English](README.en.md) · [한국어](README.ko.md) · [他の言語](../README.md)

**集中タイマー**は、Mac 向けの無料・オープンソースの集中支援アプリです。タイマーをセット（デフォルト 25 分）して、ひとつのことに取り組む時間を作ります。終了時にお知らせし、毎日の成果を記録します。任意で **3000 体以上の集中パートナー** も選べます。

アカウント登録不要。集中タイマーはオフラインで動作。データはすべてお使いの Mac に保存されます。

---

## こんな方におすすめ

- **すぐ始められる**：アプリを開いて「集中を開始」を押すだけ
- **シンプルな画面**：大きなカウントダウンリングと余白たっぷりのデザインで、気が散りにくい
- **時間は自由に調整**：デフォルト 25 分、±5 分で変更可能。設定でデフォルト時間も変更できます
- **いつでも操作可能**：一時停止、再開、途中終了に対応
- **成長が見える**：集中完了のたびに木を 1 本 🌳。過去 26 週間のヒートマップで振り返り
- **集中パートナー（任意）**：**3000+ キャラ** — 図鑑からダウンロード、デスクトップ浮窗も可能
- **完全無料**：オープンソースで、コードも公開されています

## 集中パートナー（任意）

集中は一人で頑張らなくて大丈夫。集中パートナーでは：

- **3000 体以上の図鑑**：検索・フィルターで好きなキャラをダウンロード
- **集中中にリアクション**：タイマー横に寄り添い、完了を祝ってくれる
- **デスクトップに浮かぶ**：フローティングウィンドウで常に表示
- **完全オプション**：いつでもオフにできる

## 使い方

### はじめての方

1. 下のリンクからダウンロードしてインストール
2.「アプリケーション」フォルダから **集中タイマー** を開く
3. **集中を開始** をクリックしてカウントダウン開始
4. 時間が来ると完了通知が表示されます

### 日常の使い方

| やりたいこと | 操作 |
|-------------|------|
| 集中を始める | **集中を開始** をクリック |
| 今回の時間を変える | リング横の **−** / **+**（5 分単位） |
| 休憩する | **一時停止** → 再開時は **再開** |
| 途中で終わる | **終了** |
| 集中パートナーを選ぶ | **集中パートナー** → 図鑑を閲覧 → ダウンロード |
| 浮窗をオン | **集中パートナー** → **デスクトップ浮窗** |
| デフォルト時間を変える | **設定** → **デフォルト時間** |
| 統計を見る | **統計** → 今日の記録、ヒートマップ |

### 4 つのタブ

- **集中**：タイマーと操作ボタン（ペット表示あり）
- **デスクトップペット**：図鑑の閲覧・ダウンロード・管理
- **統計**：今日の回数、ヒートマップ、日別記録
- **設定**：デフォルト時間など

## ダウンロード方法

### ステップ 1：Mac のチップを確認

1. 画面左上の **Apple メニュー** をクリック
2. **この Macについて** を選択
3. 表示を確認：
   - **チップ**（例：Apple M2）→ **Apple シリコン**
   - **プロセッサ**（例：Intel Core i5）→ **Intel Mac**

### ステップ 2：リンクをクリックしてダウンロード

最新バージョン：**1.0.0**

| チップ | 直接ダウンロード（日本語版） |
|--------|------------------------------|
| Apple シリコン（M シリーズ） | [**Apple シリコン版をダウンロード**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-ja-arm64.dmg) |
| Intel | [**Intel 版をダウンロード**](https://github.com/JackZhong1998/attentionclock/releases/download/v1.0.0/AttentionClock-1.0.0-ja-x86_64.dmg) |

または **[全バージョンのダウンロードページ](https://github.com/JackZhong1998/attentionclock/releases/latest)** で、ファイル名に `ja` を含むものを選んでください。

### ステップ 3：インストール

1. ダウンロードした `.dmg` をダブルクリック
2. **「インストールして開く」** をダブルクリック — 自動でコピーして起動
3. 「損傷しています」と出たら **「初回起動ガイド.txt」** を参照

### ステップ 4：初回起動（「損傷しています」と表示される場合）

ブラウザからダウンロードしたアプリは、macOS によりブロックされることがあります。**アプリが壊れているわけではありません。** 以下の手順を**順番どおり**に行ってください：

1. 「アプリケーション」で AttentionClock（集中タイマー）を**ダブルクリック**して開く
2. 「損傷しています」ダイアログが出たら **キャンセル** をクリック（⚠️ **ゴミ箱に入れる** は押さない）
3. **システム設定** → **プライバシーとセキュリティ** を開く
4. **セキュリティ** セクションまでスクロールし、AttentionClock の **このまま開く** をクリック
5. もう一度アプリを開いて使用する

> **順番が重要です**：手順 1・2 を先に行わないと **このまま開く** ボタンは表示されません。

## よくある質問

**インターネットは必要ですか？**  
いいえ。完全にローカルで動作します。

**データはどこに保存されますか？**  
お使いの Mac 内（`~/Library/Application Support/AttentionClock/`）のみ。外部に送信されません。

**Windows / iPhone には対応していますか？**  
現在は Mac（macOS 14.0 以降）のみです。

**ダウンロードリンクが開けない場合は？**  
[GitHub Release](https://github.com/JackZhong1998/attentionclock/releases) が公開されているかご確認ください。未公開の場合は、公開をお待ちください。

**「損傷しています」と表示される？**  
上記ステップ 4 の手順どおり：ダブルクリック → キャンセル → システム設定で「このまま開く」。
