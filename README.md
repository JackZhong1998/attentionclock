## 这是什么？

专注时钟帮你在 Mac 上设定一段不受打扰的时间（默认 25 分钟），支持暂停、继续和提前结束。每完成一次专注种一棵树 🌳，还能查看热力图和每日统计。可选像素小猫陪伴。

- 免费开源，无需注册
- 数据保存在本机，不上传云端
- 系统要求：macOS 14.0+

各语言的详细介绍、使用说明和安装步骤，请点击上表中的「说明文档」链接。

## 开发者

```bash
# 生成全部语言 Release（21 语言 × 2 架构 = 42 个 DMG）
./scripts/build-release.sh 1.0.0

# 更新翻译后重新生成 xcstrings
python3 scripts/generate-localizations.py

# 更新各语言 README
python3 scripts/generate-readmes.py
```

翻译源文件：`scripts/locales/{语言代码}.json`
