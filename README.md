## 首次打开提示「已损坏」？

从浏览器下载后，macOS 可能会拦截应用并弹出「已损坏」提示。**这不是应用真的坏了**，请**按顺序**操作：

1. 打开 `.dmg` 后，**双击「安装并打开」**
2. 若弹出对话框，点击 **取消**（⚠️ **不要**点「移到废纸篓」）
3. 打开 **系统设置** → **隐私与安全性** → **仍要打开**
4. 详见 DMG 内的 **「首次打开指南.txt」**

> **顺序很重要**：必须先完成第 1、2 步，「仍要打开」才会出现。各语言详细说明见上表中的「说明文档」。

## 这是什么？

专注时钟帮你在 Mac 上设定一段不受打扰的时间（默认 25 分钟），支持暂停、继续和提前结束。每完成一次专注种一棵树 🌳，还能查看热力图和每日统计。

**专注伙伴（可选）** 是 v1.1 的亮点：从 **3000+ 像素图鉴** 中挑选角色或伙伴，下载后在专注页陪你一起做事；开启 **桌面浮窗** 后，伙伴会常驻桌面，切换应用也不离开。专注时有表情反馈，完成还会庆祝——但完全可选，关掉也不影响计时。

- 免费开源，无需注册
- 数据保存在本机，不上传云端
- 专注伙伴：浏览/下载需联网，专注计时可离线
- 系统要求：macOS 14.0+

各语言的详细介绍、使用说明和安装步骤，请点击上表中的「说明文档」链接。

## 开发者

```bash
# 生成全部语言 Release（21 语言 × 2 架构 = 42 个 DMG）
./scripts/build-release.sh 1.1.0

# 更新翻译后重新生成 xcstrings
python3 scripts/generate-localizations.py

# 更新各语言 README
python3 scripts/generate-readmes.py

# 生成多语言官网静态页面
python3 scripts/generate-website.py
```

翻译源文件：`scripts/locales/{语言代码}.json`
